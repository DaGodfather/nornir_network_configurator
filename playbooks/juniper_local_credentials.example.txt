# playbooks/juniper_local_credentials.txt
# Juniper local credential verification entries
#
# PURPOSE:
#   Before removing TACACS from a Juniper device, this script verifies that the
#   encrypted passwords listed here already exist in the device's running config.
#   This ensures local login will work BEFORE TACACS is removed.
#
# FORMAT:
#   One JunOS 'set' command per line (as shown by 'show configuration | display set').
#   Lines starting with '#' and blank lines are ignored.
#
# HOW TO GET THE VALUES:
#   On the Juniper device, run:
#     show configuration system root-authentication | display set
#     show configuration system login user <username> | display set
#   Copy the 'set system ... encrypted-password "..."' lines below.
#
# IMPORTANT:
#   The encrypted password hashes must already be configured on the devices.
#   Use the 'update_cisco_local_credentials' equivalent for Juniper to push
#   local accounts before running make_juniper_login_local.

# Root authentication encrypted password
set system root-authentication encrypted-password "$6$abc123examplehash$..."

# Local user account(s) - add one line per user that should have local auth
set system login user netops authentication encrypted-password "$6$xyz456examplehash$..."
